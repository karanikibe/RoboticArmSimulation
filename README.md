# Robotic Arm Simulation

A robotic arm simulation built with Unity.

## Requirements
- Unity Hub: https://unity.com/download
- Unity Editor Version: 2021.x (or whichever version you used)
- Git: https://git-scm.com/download/win

## How to Run
1. Install Unity Hub from https://unity.com/download
2. Clone this repository:
```
   git clone https://github.com/karanikibe/RoboticArmSimulation.git
```
3. Open Unity Hub
4. Click **Add project from disk**
5. Navigate to the cloned folder and select **FinalRobotArm**
6. Let Unity auto-generate the Library folder (first launch takes a few minutes)
7. Press the **Play** button to run the simulation

## Project Structure
```
RoboticArmSimulation/
└── FinalRobotArm/
    ├── Assets/        # Scenes, Scripts, Prefabs
    ├── Packages/      # Package dependencies
    └── ProjectSettings/ # Unity project settings
```
